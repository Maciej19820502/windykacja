Pobierz wzorzec CSV
Wymagane kolumny:
  • Kontrahent - nazwa firmy
  • Nr faktury - numer faktury
  • Kwota - kwota (np. 5000.00)
  • Waluta - kod waluty (np. PLN, EUR, USD; domyślnie PLN)
  • Data wystawienia - format YYYY-MM-DD lub DD.MM.YYYY
  • Termin platnosci - format YYYY-MM-DD lub DD.MM.YYYY
  • Data platnosci - format YYYY-MM-DD lub DD.MM.YYYY (puste = nieopłacona)
Obsługiwane separatory:
  • Średnik (;), tabulator, przecinek
Import faktur KSeF (XML)
Informacje o KSeF XML

Importuj faktury bezpośrednio z plików XML pobranych z Krajowego Systemu e-Faktur.

Obsługiwane formaty:
  • FA(2) — schemat z 2023 r.
  • FA(3) — schemat z 2025 r.
Jak pobrać XML z KSeF:
  1. Zaloguj się do ksef.mf.gov.pl
  2. Wyszukaj faktury zakupowe
  3. Pobierz wybrane faktury jako pliki XML
  4. Wgraj pobrane pliki powyżej
